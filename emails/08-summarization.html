Summarization API: TL;DR Any Document

Drop in a wall of text, get a concise summary back. The Summarization API condenses long-form content into digestible summaries with a configurable max_length parameter.

POST /api/v1/summarization

Send any text with an optional max_length to control output size. Get a clean, coherent summary in return.

cURL Example

curl -X POST https://api.text-generator.io/api/v1/summarization \
  -H "secret: YOUR_API_SECRET" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Your long article or document text here...",
    "max_length": 150
  }'
