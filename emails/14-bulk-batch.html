Bulk Processing: Generate 1000 Texts in One Call

Need to generate hundreds or thousands of texts? The Bulk Generation API processes multiple prompts in parallel, tracks progress, and delivers results — all from a single request.

The API

POST /api/v1/generate-bulk

Send an array of prompts and get back results for all of them. Prompts are processed in parallel for maximum throughput.

BatchJob Tracking

Every bulk request creates a tracked batch job in the database. Monitor progress, retry failed items, and retrieve results at any time. No more fire-and-forget uncertainty.
