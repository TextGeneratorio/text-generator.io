Image Captioning: Understand Any Image Instantly

Send an image, get a description back. The Image Captioning API uses Microsoft's GIT-base model to generate accurate, natural-language captions for any image in seconds.

Fast vs Quality Modes

Fast Mode (default)

Quick captions optimized for speed. Ideal for high-volume processing where throughput matters most.

Quality Mode

More detailed, nuanced descriptions. Best for accessibility alt-text and content that needs precision.

cURL Example

curl -X POST https://api.text-generator.io/api/v1/image-caption \
  -H "secret: YOUR_API_SECRET" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/photo.jpg"}'
