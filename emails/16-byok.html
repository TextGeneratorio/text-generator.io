BYOK: Bring Your Own API Keys for Max Savings

Already have API contracts with OpenAI, Anthropic, or Google? With Bring Your Own Key, you can plug your existing provider keys into Text Generator and use our full platform — tools, UI, pipelines, and infrastructure — while billing goes directly to your own accounts.

How It Works

1. Add Your Key

Go to your account page and paste in your API key from OpenAI, Anthropic, or Google. You can add keys for multiple providers at once.

2. Encrypted at Rest

Every key is encrypted with Fernet symmetric encryption before storage. You'll only ever see the prefix displayed back to you (e.g. sk-...abc).

3. Use the Platform

All features — playground, prompt optimizer, research tools, bulk endpoints — work exactly the same. Requests route through your own provider keys instead of ours.
