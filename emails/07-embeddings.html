Day 18 · Embeddings

Semantic Search in 3 Lines of Code

Turn text into 768-dimension vectors with the Embeddings API. Powered by ModernBERT-base, it gives you production-ready embeddings for search, clustering, and RAG in a single POST request.

POST /api/v1/feature-extraction

Send text, get back a 768-dimensional embedding vector. Use it for similarity comparisons, clustering, or as input to your own ML models.

Python Example

import requests
import numpy as np

# Get embeddings for two texts
def embed(text):
  return requests.post(
    "https://api.text-generator.io/api/v1/feature-extraction",
    json={"text": text},
    headers={"secret": "YOUR_SECRET"}
  ).json()

a = np.array(embed("How do I reset my password?"))
b = np.array(embed("I forgot my login credentials"))

# Cosine similarity
similarity = np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))
print(f"Similarity: {similarity:.3f}")  # ~0.92
