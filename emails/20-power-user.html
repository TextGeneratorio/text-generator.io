Power User Guide: Tips, Tricks & What's Next

You've been with us for 90 days — and you now have access to one of the most versatile AI platforms out there. Here are the pro tips that separate casual users from power users.

Pro Tips

Tip 1

Use temperature and top_p together to control creativity vs consistency. Low temperature (0.1-0.3) for factual tasks, higher (0.7-1.0) for creative writing.

Tip 2

Set max_length and stop_sequences for precise output control. Stop on newlines for single-line answers, or set max_length to cap token spend.

Tip 3

Use the OpenAI-compatible endpoint for easy SDK integration. Change one line — the base URL — and your existing OpenAI code works with our models instantly.

Tip 4

Combine TTS + text generation for audio content pipelines. Generate a script, then convert it to speech — perfect for podcasts, narration, and accessibility.

Tip 5

Use embeddings + generation for RAG workflows. Embed your documents, retrieve the most relevant chunks, then pass them as context to the generation API for grounded answers.

Tip 6

Batch process with the bulk endpoint for high-volume needs. Send up to 100 prompts in a single request and get results back in parallel — far faster than sequential calls.

Tip 7

Self-host for full data sovereignty. Run the inference server on your own infrastructure when compliance requires it — your data never leaves your network.
