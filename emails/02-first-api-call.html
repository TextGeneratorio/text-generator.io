Your First API Call in 30 Seconds

Let's generate your first text. Pick your language — here's the same call in cURL, Python, and JavaScript.

cURL

curl -X POST https://api.text-generator.io/api/v1/generate \
  -H "secret: YOUR_API_SECRET" \
  -H "Content-Type: application/json" \
  -d '{"text": "Write a product description for", "max_length": 200}'

Python

import requests

resp = requests.post(
  "https://api.text-generator.io/api/v1/generate",
  json={"text": "Write a tagline for", "max_length": 100},
  headers={"secret": "YOUR_API_SECRET"}
)
print(resp.json())

JavaScript

const resp = await fetch("https://api.text-generator.io/api/v1/generate", {
  method: "POST",
  headers: { "secret": "YOUR_SECRET", "Content-Type": "application/json" },
  body: JSON.stringify({ text: "Generate a haiku about", max_length: 50 })
})

Your API secret is on your account page. It's all you need to authenticate — no OAuth, no tokens to refresh.
