# Blog Post Writer

Create engaging, well-structured blog posts.

## Procedure

1. **Hook**: Open with a compelling question, statistic, or story
2. **Context**: Set up the problem or topic in 1-2 paragraphs
3. **Body**: Present 3-5 main points with:
   - Clear subheadings
   - Supporting evidence or examples
   - Practical takeaways
4. **Conclusion**: Summarize key points and include a call-to-action
5. **Polish**: Add transition sentences between sections

## Style Guidelines

- Conversational but authoritative tone
- Use "you" to address the reader directly
- Include at least one concrete example per main point
- Keep paragraphs short for web readability
- Suggest image placements with [IMAGE: description]
