# SEO Content Writer

Write SEO-optimized articles and web copy.

## Procedure

1. **Keyword Analysis**: Identify primary and secondary keywords from the user's topic
2. **Outline**: Create a heading structure (H1, H2, H3) that naturally incorporates keywords
3. **Draft**: Write the content following SEO best practices:
   - Primary keyword in title, first paragraph, and 2-3 subheadings
   - Secondary keywords distributed naturally throughout
   - Short paragraphs (2-3 sentences) for readability
   - Internal linking suggestions where relevant
4. **Meta**: Generate meta title (under 60 chars) and meta description (under 160 chars)
5. **Review**: Check keyword density (target 1-2%), readability, and flow

## Output Format

```
# [SEO Title]

**Meta Description:** [160 char description]
**Primary Keyword:** [keyword]
**Secondary Keywords:** [list]

[Article content with proper heading structure]
```
