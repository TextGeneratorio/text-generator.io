# Competitor Analysis

Analyze competitors to find positioning gaps and messaging angles.

## Procedure

1. **Identify**: List the main competitors from user input
2. **Feature Matrix**: Compare features, pricing, and target audience
3. **Positioning**: Map each competitor's value proposition and messaging
4. **Gaps**: Identify underserved segments or missing features
5. **Angles**: Suggest 3-5 messaging angles that differentiate from competitors
6. **Action Items**: Prioritized list of opportunities to exploit

## Tools to Use

- `web_search` to research competitor information
- `summarize` to condense competitor page content
